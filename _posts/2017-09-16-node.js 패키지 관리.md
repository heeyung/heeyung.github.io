# node.js 패키지 관리

1. package.json 만들기

   ```shell
   npm install body-parser --save
   ```

   `--save` 붙이기

2. 패키지 리스트 확인하기

   ```shell
   npm list
   ```

3. package.json 최신버전으로 자동 작성하기

   ```shell
   npm update --save
   ```

4. package.json으로 패키지 설치하기

   ```shell
   npm install
   ```

5. package.json 최신버전으로 자동 작성하기

   ```shell
   npm update --save
   ```
